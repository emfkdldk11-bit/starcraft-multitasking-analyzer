# 🚀 StarCraft Multitasking Analyzer (v1.0)

이 프로그램은 스타크래프트 리플레이를 분석하여 사용자의 **멀티태스킹 지수**를 산출하는 데이터 분석 도구입니다.

---

## 📊 주요 분석 지표

* **EAPM (Effective APM):** 단순 연타(Spam)를 제외한 실제 유효 조작 속도를 측정합니다.
* **Macro vs Micro:** 시계열 데이터를 통해 운영(생산)과 교전 컨트롤의 비중을 실시간으로 비교합니다.
* **Multitasking Score:** **$\sqrt{Micro \times Macro}$** 로직을 사용하여, 교전 중에도 얼마나 운영을 놓지 않았는지를 객관적 점수로 환산합니다.

---

## 🛠️ 시작하기 (How to Use)

1. **준비물:** 리플레이 분석 엔진인 `screp.exe` 파일이 프로그램과 같은 폴더에 있어야 합니다.
2. **실행:** `StarAnalyzer.exe`를 실행합니다.
3. **분석:** 분석하고 싶은 스타크래프트 리플레이(`.rep`) 파일을 창 위로 드래그 앤 드롭 하세요!

## 💡 개발자의 철학
> "단순히 손이 빠른 것보다, 그 속도가 어디에 쓰이고 있는지를 아는 것이 실력 향상의 지름길입니다."
